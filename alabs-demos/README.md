# A://Labs Interactive Demos

> **Standalone, offline-capable HTML course modules.**
>
> Open `index.html` in any browser — no server, no build step, no internet required after initial font load.

## What's Here

Self-contained interactive HTML modules generated directly from the Allternit codebase. Each file includes:

- Full curriculum content with real code examples
- Interactive quizzes with instant feedback
- Canvas animations (DAG visualizations, scheduler simulations, circuit breakers)
- Scroll-reveal animations and progress tracking
- Syntax-highlighted code blocks
- Capstone project descriptions

## How to Use

1. **Browse the index:** Open `index.html` for a categorized grid view
2. **Open any module:** Click a module card → opens the standalone HTML file
3. **Works offline:** After initial load, no network calls needed (except Google Fonts on first visit)

## How to Update

The demo index is auto-generated. To rebuild:

```bash
cd /Users/macbook/Desktop/allternit-workspace/allternit
npx tsx scripts/generate-demo-index.ts
```

This scans `alabs-generated-courses/` and `alabs-demos/` and regenerates `index.html`.

## How Modules Get Here

```
alabs-generated-courses/     ← Generated by agent swarms
         │
         ▼ cp
    alabs-demos/             ← Copied here for distribution
         │
         ▼ npx tsx scripts/generate-demo-index.ts
    alabs-demos/index.html   ← Auto-generated landing page
```

## Source

All modules are generated from the actual Allternit codebase:
- `packages/@allternit/plugin-sdk/`
- `packages/@allternit/workflow-engine/`
- `packages/@allternit/provider-adapters/`
- `surfaces/allternit-platform/` (for bridge modules)
