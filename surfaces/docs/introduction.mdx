## What is Allternit?

Allternit (Allternit) is an enterprise-grade agentic operating system designed to orchestrate intelligent agents across multiple interfaces. Built on the principle of **"One Brain, Multiple Faces, One Nervous System,"** Allternit provides a unified platform for developing, deploying, and managing AI agents at scale.

<Note>
This documentation reflects the current Phase 1 Refactor. Some APIs may change as we consolidate from 7-apps/ to surfaces/.
</Note>

## Installation

Install the Allternit CLI globally:

<CodeGroup>
```bash npm
npm install -g @allternit/cli
```

```bash yarn
yarn global add @allternit/cli
```

```bash pnpm
pnpm add -g @allternit/cli
```
</CodeGroup>

Or use the one-line installer for the BYOC runtime:

```bash
curl -fsSL https://install.allternit.com | bash
```
