## Overview

Gizzi is the core runtime engine that executes agent code. It provides a secure sandboxed environment for running agent logic while managing state, memory, and communication with external services.

## Execution Model

Gizzi uses a message-passing execution model where agents communicate through channels. Each agent runs in its own isolated context with access to:

- **Memory store** - State persistence across executions
- **Skill registry** - Capability discovery and invocation
- **Message bus** - Inter-agent communication
- **Model gateway** - LLM interactions

```typescript
import { Agent } from '@allternit/core';

const agent = new Agent({
  id: 'my-agent',
  runtime: 'gizzi-pro',
  skills: ['code', 'web', 'file'],
});

await agent.initialize();
const result = await agent.run('Analyze this codebase');
```

## Sandboxing

Gizzi provides multiple sandboxing levels:

| Level | Description | Use Case |
|-------|-------------|----------|
| `none` | No restrictions | Trusted environments only |
| `soft` | File system restrictions | Development |
| `strict` | Full sandbox with network isolation | Production |
| `container` | Docker container isolation | Maximum security |

```typescript
const agent = new Agent({
  id: 'secure-agent',
  sandbox: 'strict',
  allowedPaths: ['/workspace'],
  network: false,
});
```

## Memory Management

Agents can persist memory across executions:

```typescript
// Store in agent memory
await agent.memory.set('user_preference', 'dark_mode');

// Retrieve later
const preference = await agent.memory.get('user_preference');

// Clear memory
await agent.memory.clear();
```

Memory types:
- **Session** - Lost when agent stops
- **Persistent** - Stored in database
- **Shared** - Accessible across agents

## Model Integration

Gizzi supports multiple LLM providers:

```typescript
const agent = new Agent({
  model: {
    provider: 'openai',
    model: 'gpt-4',
    temperature: 0.7,
  },
});
```

Supported providers:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Local (Ollama, LocalAI)
