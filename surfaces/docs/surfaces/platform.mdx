## Overview

`allternit-platform` is the main web-based UI surface. It is a Next.js 15 application that serves as the shell for all agent interactions, workflows, and tooling. It runs on **Turbopack** in development for fast, low-memory compilation.

## Local Development

```bash
cd surfaces/allternit-platform
pnpm dev        # starts on http://localhost:3013 (Turbopack)
pnpm dev:clean  # clears .next cache first, then starts
```

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router) |
| Bundler (dev) | Turbopack |
| Auth | Clerk (with stub fallback) |
| Database | SQLite via Prisma + Drizzle |
| State | Zustand |
| AI SDK | Vercel AI SDK |

## Environment

Requires a `.env.local` file in `surfaces/allternit-platform/`:

```env
DATABASE_URL=file:./dev.db
# Optional: Clerk auth keys
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...
```

## Port

Always runs on **port 3013**. The Electron desktop app loads this URL when the platform server is running.
