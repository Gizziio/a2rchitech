Welcome to Allternit

Allternit (Allternit) is an enterprise-grade agentic operating system designed to orchestrate intelligent agents across multiple interfaces. Built on the principle of "One Brain, Multiple Faces, One Nervous System."

Phase 1 Refactor

This documentation reflects the current Phase 1 Refactor. Some APIs may change as we consolidate from 7-apps/ to surfaces/.

Core Components

One Brain (Gizzi)

The core runtime engine that plans, reasons, and executes agent logic.

Multiple Faces (Surfaces)

Desktop, Web, CLI, and Browser Extension interfaces for different use cases.

One Nervous System (SDK)

A unified interface connecting all surfaces to the brain.

Quickstart

Install the CLI

npm install -g @allternit/cli

Or use the one-line installer for BYOC:

curl -fsSL https://install.allternit.com | bash

Create your first project

allternit init my-first-agent
cd my-first-agent
allternit dev

Architecture

The Allternit platform follows a unique architectural pattern that separates the core intelligence (Brain) from the user interfaces (Faces) while maintaining a unified communication layer (Nervous System).

Control Plane vs Data Plane

┌─────────────────┐         ┌─────────────────────┐
│  Control Plane  │         │     Data Plane      │
│  (Vercel)       │◄───────►│  (Your VPS/Server)  │
│                 │  HTTPS  │                     │
│  • Platform UI  │         │  • Runtime Agent    │
│  • API Gateway  │         │  • SQLite DB        │
│  • Agent Reg.   │         │  • Worker Engine    │
└─────────────────┘         └─────────────────────┘
       ▲                               ▲
       │                               │
       │    WebSocket (Direct)         │
       └───────────────────────────────┘
            Browser → Runtime Agent

BYOC Runtime

BYOC (Bring Your Own Cloud) lets you deploy the Allternit runtime agent to your own infrastructure. Your data never leaves your servers.

Installation

curl -fsSL https://install.allternit.com | bash

# Or download manually
wget https://dl.allternit.com/stable/latest/allternit-runtime-linux-amd64
chmod +x allternit-runtime
sudo mv allternit-runtime /usr/local/bin/

Configuration

Create /etc/allternit/runtime.yaml:

server:
  host: 0.0.0.0
  port: 8080

auth:
  api_key: ${ALLTERNIT_API_KEY}

control_plane:
  endpoint: https://api.allternit.com

storage:
  path: /var/lib/allternit/data.sqlite

Authentication

All API requests require authentication using an API key:

curl -H "Authorization: Bearer allternit_live_xxxxxxxxxxxxxxxx" \
  https://api.allternit.com/v1/agents

API Endpoints

Base URL: https://api.allternit.com/v1

Agents

GET    /agents           # List agents
POST   /agents           # Create agent
GET    /agents/{id}      # Get agent
POST   /agents/{id}/run  # Run agent

Sessions

GET    /sessions/{id}           # Get session
GET    /sessions/{id}/messages  # List messages

TypeScript SDK

npm install @allternit/sdk
import { AllternitClient } from '@allternit/sdk';

const client = new AllternitClient({
  apiKey: process.env.ALLTERNIT_API_KEY,
  endpoint: 'https://api.allternit.com',
});

const agent = await client.agents.create({
  name: 'my-agent',
  runtime: 'gizzi-pro',
});

const result = await client.agents.run(agent.id, {
  input: 'Analyze this data',
});
