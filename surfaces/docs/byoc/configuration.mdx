## Configuration File

The runtime uses a YAML configuration file located at `/etc/allternit/runtime.yaml`:

```yaml
# Server configuration
server:
  host: 0.0.0.0
  port: 8080
  tls:
    enabled: false
    cert: /path/to/cert.pem
    key: /path/to/key.pem

# Authentication
auth:
  api_key: ${ALLTERNIT_API_KEY}
  allowed_origins:
    - https://platform.gizziio.com
    - https://app.allternit.com

# Control plane connection
control_plane:
  endpoint: https://api.allternit.com
  heartbeat_interval: 30
  
# Database
storage:
  path: /var/lib/allternit/data.sqlite
  backup_interval: 3600

# Logging
logging:
  level: info  # debug, info, warn, error
  format: json  # json, text
  output: stdout  # stdout, file, both
  file: /var/log/allternit/runtime.log

# Limits
limits:
  max_connections: 100
  max_message_size: 10485760  # 10MB
  request_timeout: 30

# Features
features:
  websockets: true
  sse: true
  webhooks: true
```

## Environment Variables

You can also use environment variables (they override config file values):

```bash
export ALLTERNIT_API_KEY=your_api_key
export ALLTERNIT_SERVER_PORT=8080
export ALLTERNIT_STORAGE_PATH=/data/allternit.db
export ALLTERNIT_LOG_LEVEL=debug
```

## Authentication

### API Key

Get your API key from [platform.gizziio.com](https://platform.gizziio.com/settings/api-keys):

```yaml
auth:
  api_key: allternit_live_xxxxxxxxxxxxxxxx
```

### CORS Origins

Specify which origins can connect to your runtime:

```yaml
auth:
  allowed_origins:
    - https://platform.gizziio.com
    - http://localhost:3000  # Development
```

## Storage

The runtime uses SQLite by default. For production, consider:

```yaml
storage:
  path: /var/lib/allternit/data.sqlite
  backup_interval: 3600  # Backup every hour
  backup_retention: 7    # Keep 7 backups
```

Backup location: `/var/lib/allternit/backups/`

## TLS/SSL

For production, enable TLS:

```yaml
server:
  tls:
    enabled: true
    cert: /etc/letsencrypt/live/your-domain.com/fullchain.pem
    key: /etc/letsencrypt/live/your-domain.com/privkey.pem
```

Or use a reverse proxy (nginx, Caddy, Traefik).

## WebSocket Configuration

WebSockets enable real-time communication:

```yaml
features:
  websockets: true
  
limits:
  max_connections: 100
  websocket:
    ping_interval: 30
    pong_timeout: 10
```

## Logging Levels

<ParamField path="logging.level" type="string">
  - `debug` - Verbose output for troubleshooting
  - `info` - Normal operational messages
  - `warn` - Warning conditions
  - `error` - Error conditions only
</ParamField>

## Validation

Validate your configuration:

```bash
allternit-runtime --config /etc/allternit/runtime.yaml --validate
```

## Reload Configuration

Restart the service to apply changes:

```bash
sudo systemctl restart allternit-runtime
```

Or send SIGHUP for graceful reload (some settings):

```bash
sudo kill -HUP $(pgrep allternit-runtime)
```
