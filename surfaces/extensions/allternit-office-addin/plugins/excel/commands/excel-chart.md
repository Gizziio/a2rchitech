# Command: excel:chart

Create a chart from the active selection or a specified data range.

## Triggers

- "create a chart"
- "visualize this data"
- "plot X vs Y"
- "make a graph of"
- "chart the sales data"
- "show me a trend line for"

## Steps

1. **Identify the data range** — use the current selection if available; otherwise ask the user to specify or default to the used range.
2. **Choose chart type** — apply the decision logic below based on the shape and nature of the data.
3. **Apply from `skills/chart-creation.md`** — use the chart creation patterns defined there for formatting, titles, and series configuration.
4. **Position below used range** — place the chart so it starts two rows below the last data row, at column A, to avoid overlapping data.
5. **Set title and axis labels** — derive from column headers where possible.

## Chart Type Decision Logic

| Data Pattern | Recommended Chart Type |
|--------------|----------------------|
| One date/time column + one or more numeric columns | Line (`Excel.ChartType.line`) |
| Categorical comparison (few categories, one metric) | Clustered Column (`Excel.ChartType.columnClustered`) |
| Multiple series comparison | Grouped Bar (`Excel.ChartType.barClustered`) |
| Part-to-whole (single series, 5 or fewer segments) | Pie (`Excel.ChartType.pie`) |
| Part-to-whole (single series, 6+ segments or wants modern look) | Doughnut (`Excel.ChartType.doughnut`) |
| Two numeric variables (correlation/scatter) | Scatter (`Excel.ChartType.xyscatter`) |
| Distribution / frequency | Column Clustered |
| Cumulative or waterfall-style | Use column clustered with stacked effect |

When in doubt, default to clustered column.

## Office.js Code Pattern

```typescript
await Excel.run(async (context) => {
  const sheet = context.workbook.worksheets.getActiveWorksheet();

  // Determine data range
  const dataRange = sheet.getRange("A1:C13"); // replace with detected or user-specified range

  // Get used range to know where to place the chart
  const usedRange = sheet.getUsedRange();
  usedRange.load("rowCount");
  await context.sync();

  const chartTopRow = usedRange.rowCount + 2; // two rows below data
  const chartLeft = 0;    // column A offset in points (approximate)
  const chartTop = chartTopRow * 20; // approximate row height in points

  // Create the chart
  const chart = sheet.charts.add(
    Excel.ChartType.line,   // swap based on decision logic
    dataRange,
    Excel.ChartSeriesBy.columns
  );

  // Position and size
  chart.setPosition(`A${chartTopRow}`, `H${chartTopRow + 20}`);
  chart.width = 480;
  chart.height = 300;

  // Titles
  chart.title.text = "Revenue Trend";
  chart.title.visible = true;

  // Axis labels
  chart.axes.valueAxis.title.text = "Value";
  chart.axes.valueAxis.title.visible = true;
  chart.axes.categoryAxis.title.text = "Period";
  chart.axes.categoryAxis.title.visible = true;

  // Legend
  chart.legend.visible = true;
  chart.legend.position = Excel.ChartLegendPosition.bottom;

  // Remove gridlines for cleaner look (optional)
  chart.axes.valueAxis.majorGridlines.visible = false;

  await context.sync();

  return { chartName: chart.name };
});
```

## Scatter Plot Pattern (Correlation)

```typescript
await Excel.run(async (context) => {
  const sheet = context.workbook.worksheets.getActiveWorksheet();
  const dataRange = sheet.getRange("B2:C50"); // X in col B, Y in col C

  const chart = sheet.charts.add(
    Excel.ChartType.xyscatter,
    dataRange,
    Excel.ChartSeriesBy.columns
  );

  chart.title.text = "Correlation: X vs Y";
  chart.title.visible = true;
  chart.setPosition("A30", "H50");
  chart.width = 480;
  chart.height = 300;

  await context.sync();
});
```

## Notes

- Always load the used range row count before positioning so the chart does not overlap data.
- For time-series data, ensure the date/time column is the first column in the range so Excel assigns it to the category axis automatically.
- If the user does not specify a range and the selection contains more than one table or disconnected regions, ask the user to clarify the exact range.
- Chart names auto-generated by Excel (e.g., "Chart 1") are returned so the user can reference or rename them.
- After creation, offer to run `excel:format` to apply consistent styling to the chart background and fonts.
