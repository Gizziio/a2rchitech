## Time Budget

| Step | Time | UX |
|------|------|-----|
| Check local backend | 100ms | Instant |
| Splash screen appear | 200ms | Instant |
| Download backend (50MB @ 10MB/s) | 5s | "Setting up..." |
| Extract backend | 1s | Progress bar |
| Start backend | 2s | "Starting..." |
| Load Platform UI | 1s | Loading spinner |
| **Total First Launch** | **~10s** | Tolerable |
| **Total Second Launch** | **<2s** | Instant |

## Error Handling

```
Download Failed:
  в””в”Җв”Җ Retry 3x with exponential backoff
      в””в”Җв”Җ Still fails вҶ’ Show: "Connection required"
          в””в”Җв”Җ Offer: "Use offline installer" or "Enter remote URL"

Backend Won't Start:
  в””в”Җв”Җ Show logs
      в””в”Җв”Җ Offer: "Reset backend" or "Report issue"

Port 4096 in Use:
  в””в”Җв”Җ Try 4097, 4098, ...
      в””в”Җв”Җ Update config dynamically

Remote Backend Unreachable:
  в””в”Җв”Җ Show: "Cannot connect to server"
      в””в”Җв”Җ Offer: "Try again" or "Use local backend"
```
