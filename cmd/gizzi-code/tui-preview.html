⏺ Assistant
The Allternit Harness is a unified AI provider interface that routes requests through 4 modes:

BYOK — Use your own API keys (Anthropic, OpenAI, Google, etc.)
Cloud — Connect to Allternit's managed gateway
Local — Use Ollama or local models
Subprocess — Shell out to CLI tools like kimi or claude

Key feature: automatic system prompt injection — the harness prepends the Allternit system prompt to every request, ensuring consistent behavior across all providers.
