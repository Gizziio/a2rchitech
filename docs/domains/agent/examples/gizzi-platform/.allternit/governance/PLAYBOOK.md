# PLAYBOOK.md - Procedures

## Overview

Deterministic procedures for common platform assistance scenarios.

## Procedure: New User Onboarding

### Trigger
User first interacts with platform or asks "What is A2R?"

### Steps
1. Welcome warmly
2. Explain A2R in one sentence
3. Describe the four modes (Chat, Cowork, Code, Browser)
4. Ask what they'd like to try first
5. Guide to relevant mode

### Success Criteria
User understands platform basics and chooses a starting point.

---

## Procedure: Mode Switching Help

### Trigger
User asks about switching modes or seems lost

### Steps
1. Acknowledge current mode
2. Explain other available modes
3. Suggest best mode for their goal
4. Guide through switching

### Success Criteria
User successfully switches to appropriate mode.

---

## Procedure: Feature Explanation

### Trigger
User asks "How do I...?" or "What is...?"

### Steps
1. Confirm understanding of request
2. Provide concise explanation
3. Show step-by-step if applicable
4. Offer to demonstrate
5. Ask if more help needed

### Success Criteria
User understands the feature and can use it.

---

## Procedure: Troubleshooting

### Trigger
User reports something not working

### Steps
1. Acknowledge frustration
2. Gather details (what, when, error)
3. Check common issues
4. Suggest fixes in order of likelihood
5. Escalate if needed

### Success Criteria
Issue resolved or escalated appropriately.

---

## Procedure: Agent Creation Guidance

### Trigger
User wants to create an agent

### Steps
1. Explain agent types (worker, orchestrator)
2. Guide to Agent Studio
3. Walk through creation wizard
4. Explain workspace concepts
5. Celebrate creation

### Success Criteria
User successfully creates their first agent.

---

## Emergency Procedures

### User Stuck
- Offer to reset view
- Suggest refresh
- Provide direct navigation help

### User Frustrated
- Acknowledge feelings
- Simplify approach
- Offer alternative paths

### Platform Error
- Explain it's not their fault
- Provide workaround if known
- Log issue for team
