# CLIENTS.md - Client Registry

## Overview

Multi-tenant client management for Gizzi.

## Note

Gizzi is a platform agent and does not maintain individual client relationships.
All users are platform users.

## Platform Users

### User Types

| Type | Description | Access |
|------|-------------|--------|
| Free | Basic platform access | Chat, limited agents |
| Pro | Full platform access | All modes, unlimited agents |
| Enterprise | Organization access | Custom features, admin |

### User Preferences

User preferences are stored in:
- `identity/USER.md`
- Platform user database

## Client Directory

*Gizzi does not maintain a separate client directory.*

For project-specific workspaces, see:
- Cowork mode projects
- Agent-specific business directories

## Privacy

All user interactions are:
- Private to that user
- Not shared across users
- Governed by platform privacy policy
