Module 3

Supervised Learning

⏱️ Estimated time: 6-7 hours

🎯 Learning Objectives

By the end of this module, you will be able to:

  • Implement linear and logistic regression from scratch and with scikit-learn
  • Understand and apply decision trees for classification and regression
  • Explain ensemble methods: Random Forests and Gradient Boosting
  • Compare algorithm performance and select appropriate methods for different problems
  • Apply regularization techniques to prevent overfitting

📚 Topics Covered

3.1 Linear Regression
3.2 Logistic Regression
3.3 Decision Trees
3.4 Ensemble Methods

📋 Prerequisites

Completion of Modules 1-2. Solid understanding of Python, NumPy, and basic ML workflow. Familiarity with calculus (derivatives) is helpful but not required.
