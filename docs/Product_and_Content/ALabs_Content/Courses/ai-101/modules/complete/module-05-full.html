Seeing Through Neural Networks

Convolutional Neural Networks (CNNs) are the dominant architecture for computer vision. Unlike standard neural networks that treat input as a flat vector, CNNs preserve spatial relationships in images through the convolution operation—scanning filters across the image to detect features like edges, textures, and shapes.

🔍 The Convolution Operation

A convolution slides a small filter (kernel) across the image, computing dot products at each position. This detects local patterns regardless of their position in the image—a property called translation invariance.

CNN Architecture Components

🔄 Convolutional Layer

Applies learnable filters to detect features. Early layers detect edges and textures; deeper layers combine these into complex patterns like eyes, wheels, or text.

📉 Pooling Layer

Reduces spatial dimensions (typically 2x) by taking max or average values. Decreases computation and provides translation invariance.

🔗 Fully Connected Layer

Traditional neural network layer at the end that combines all features to make the final classification decision.

Building a CNN with TensorFlow 

import tensorflow as tf
from tensorflow.keras import layers, models

# Build CNN architecture
model = models.Sequential([
    # First convolutional block
    layers.Conv2D(32, (3, 3), activation='relu', 
                  input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    
    # Second convolutional block
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    
    # Flatten and classify
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])
