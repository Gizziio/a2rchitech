## Learning Outcomes

Upon successful completion, students will be able to:

1. **Explain AI Fundamentals** - Define AI, ML, and deep learning relationships
2. **Implement ML Algorithms** - Build supervised learning models
3. **Design Neural Networks** - Create networks from scratch using modern frameworks
4. **Apply Computer Vision** - Use CNNs for image understanding
5. **Work with Language Models** - Implement NLP with transformers
6. **Evaluate Models** - Select metrics and validate performance
7. **Deploy AI Systems** - Productionize models effectively
8. **Build Agentic Systems** - Create autonomous agents for the allternit protocol

## Module Breakdown

### MODULE 1: Foundations of AI
**Duration:** 4-5 hours  
**Prerequisites:** None

#### 1.1 What is Artificial Intelligence?
- Defining AI, ML, and deep learning
- The AI hierarchy (nested relationship)
- Real-world applications in agentic systems
- Knowledge check: Concept identification

#### 1.2 History of AI
- The Turing Test (1950)
- Dartmouth Workshop (1956)
- AI summers and winters
- Deep learning revolution (2012-present)
- Knowledge check: Timeline comprehension

#### 1.3 Types of AI & ML
- ANI, AGI, ASI
- Supervised learning
- Unsupervised learning
- Reinforcement learning
- Knowledge check: Type classification

#### 1.4 Environment Setup
- Python installation
- Google Colab setup
- Required libraries overview
- A://labs integration

#### Assessment
- **Discussion:** Protocol Introductions
- **Quiz:** 10 questions (20 minutes)

---

### MODULE 2: Machine Learning Basics
**Duration:** 5-6 hours  
**Prerequisites:** Module 1

#### 2.1 The ML Workflow
- 6-step process overview
- Iterative nature of ML
- Common pitfalls
- Agentic system considerations

#### 2.2 Data Collection & Preprocessing
- Data quality issues
- Missing values handling
- Feature engineering
- Scaling and normalization

#### 2.3 Train/Validation/Test Splits
- Why we split data
- Stratified sampling
- Cross-validation
- Data leakage prevention

#### 2.4 Evaluation Metrics
- Classification: Accuracy, Precision, Recall, F1
- Regression: MSE, RMSE, R²
- Confusion matrices
- Choosing appropriate metrics

#### Assessment
- **Lab:** Iris Dataset Exploration (Jupyter Notebook)
- **Discussion:** Real-world ML Applications
- **Quiz:** 10 questions (20 minutes)

---

### MODULE 3: Supervised Learning
**Duration:** 6-7 hours  
**Prerequisites:** Module 2

#### 3.1 Linear Regression
- Mathematics of linear regression
- Ordinary Least Squares (OLS)
- Multiple regression
- Code implementation with scikit-learn

#### 3.2 Logistic Regression
- Binary classification
- Sigmoid function
- Decision boundaries
- Multiclass extension

#### 3.3 Decision Trees
- Tree structure and splits
- Information gain
- Gini impurity
- Overfitting in trees

#### 3.4 Ensemble Methods
- Random Forest
- Gradient Boosting (XGBoost)
- Voting classifiers
- When to use ensembles

#### Assessment
- **Assignment:** Housing Price Prediction (100 pts)
- **Discussion:** Interpretability vs Performance
- **Quiz:** 12 questions (25 minutes)

---

### MODULE 4: Neural Networks
**Duration:** 7-8 hours  
**Prerequisites:** Module 3

#### 4.1 Biological Inspiration
- The biological neuron
- Artificial neuron analogy
- Historical context

#### 4.2 The Perceptron
- Single neuron model
- Implementation from scratch
- Linear separability
- The XOR problem

#### 4.3 Multi-Layer Networks
- Hidden layers
- Forward propagation
- Network architecture design

#### 4.4 Backpropagation
- Chain rule application
- Gradient computation
- Weight updates
- Implementation walkthrough

#### 4.5 Activation Functions
- Sigmoid, Tanh, ReLU
- Leaky ReLU, ELU, Swish
- Choosing activations

#### 4.6 TensorFlow & Keras
- High-level API overview
- Building sequential models
- Training and evaluation
- Saving and loading

#### Assessment
- **Lab:** Neural Network from Scratch (75 pts)
- **Assignment:** Digit Recognition (100 pts)
- **Discussion:** The Black Box Problem
- **Quiz:** 15 questions (30 minutes)

---

### MODULE 5: Deep Learning
**Duration:** 8-9 hours  
**Prerequisites:** Module 4

#### 5.1 The Deep Learning Revolution
- Why now? (data, compute, algorithms)
- AlexNet breakthrough
- Modern applications
- Agentic system implications

#### 5.2 CNNs for Computer Vision
- Convolution operation
- Filters and feature maps
- Pooling layers
- Architecture patterns (LeNet, VGG, ResNet)

#### 5.3 RNNs for Sequences
- Sequential data handling
- LSTM and GRU
- Bidirectional RNNs
- Applications in NLP

#### 5.4 Regularization Techniques
- Dropout
- Batch normalization
- Data augmentation
- Early stopping

#### 5.5 Transfer Learning
- Pre-trained models
- Fine-tuning strategies
- Feature extraction
- When to use transfer learning

#### 5.6 PyTorch Introduction
- Tensor operations
- Autograd
- Model definition
- Training loops

#### Assessment
- **Lab:** CNN Image Classification (100 pts)
- **Assignment:** Sentiment with RNNs (100 pts)
- **Discussion:** Deep Learning Applications
- **Quiz:** 15 questions (30 minutes)

---

### MODULE 6: NLP & Transformers
**Duration:** 8-10 hours  
**Prerequisites:** Module 5

#### 6.1 NLP Fundamentals
- Text preprocessing
- Tokenization
- Stop words and stemming
- Text representation

#### 6.2 Word Embeddings
- One-hot encoding limitations
- Word2Vec (CBOW, Skip-gram)
- GloVe
- Using pre-trained embeddings

#### 6.3 Attention Mechanisms
- The attention intuition
- Query, Key, Value
- Self-attention
- Multi-head attention

#### 6.4 The Transformer Architecture
- "Attention Is All You Need"
- Encoder-decoder structure
- Positional encoding
- Architecture advantages

#### 6.5 BERT & Modern NLP
- BERT architecture
- Fine-tuning BERT
- Hugging Face ecosystem
- GPT and generative models

#### 6.6 Generative AI & Ethics
- LLM capabilities and limitations
- Bias in AI systems
- Responsible AI development
- Future of agentic systems

#### Assessment
- **Lab:** Fine-tuning BERT (100 pts)
- **Agentic Capstone:** Build an AI Application (250 pts)
- **Discussion:** AI Ethics & Responsibility
- **Quiz:** 15 questions (30 minutes)

## Capstone Project Details

### Overview
Students build a complete AI application demonstrating skills from the course.

### Track Options
1. **Computer Vision** - Image classifier, object detector, or medical imaging system
2. **NLP** - Sentiment analyzer, chatbot, or text summarizer
3. **Predictive Analytics** - Forecasting model, recommendation system
4. **Agentic System** - Autonomous agent for a specific protocol task

### Deliverables
- Code repository (GitHub)
- Technical report (5-10 pages)
- Demo video (3-5 minutes)
- Presentation (10 minutes)

### Timeline
- Week 1: Proposal
- Week 2: Data collection
- Week 3: Initial model
- Week 4: Refinement
- Week 5: Documentation
- Week 6: Submission
