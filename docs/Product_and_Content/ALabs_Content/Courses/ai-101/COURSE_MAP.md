## Quick Start for Instructors

1. **Review Content**
   ```bash
   cd ai-ml-course
   ls -la modules/
   ```

2. **Import to Canvas**
   ```bash
   npm install @playwright/test
   node canvas-import.js --email YOUR_EMAIL --password YOUR_PASS --courseId COURSE_ID
   ```

3. **Customize**
   - Update home-page.html with your institution branding
   - Modify syllabus.html with your policies
   - Adjust deadlines in final-project.md

## Quick Start for Students
