Course Description

This comprehensive course provides an immersive introduction to Artificial Intelligence and Machine Learning within the context of agentic systems. You'll explore theoretical foundations while building real projects using industry-standard tools and frameworks.

By the end of this course, you'll understand how to build intelligent agents that can perceive their environment, learn from data, make decisions, and operate within the allternit protocol. This is your foundation for contributing to the future of autonomous, agentic computing.

Learning Outcomes

By the end of this course, you will be able to:

✓ Explain AI, ML, and deep learning fundamentals and their relationships
✓ Implement supervised and unsupervised learning algorithms
✓ Design and train neural networks for various tasks
✓ Apply computer vision and NLP techniques to real problems
✓ Evaluate model performance using appropriate metrics
✓ Deploy ML models to production environments
✓ Build agentic systems within the allternit protocol
✓ Discuss ethical AI development and responsible deployment

Module Structure

Module 1: Foundations of AI

4-5 hours

History of AI, types of AI (ANI, AGI, ASI), the ML landscape, supervised/unsupervised/reinforcement learning, and setting up your allternit development environment.

Module 2: Machine Learning Basics

5-6 hours

The ML workflow, data collection and preprocessing, train/validation/test splits, evaluation metrics (accuracy, precision, recall, F1), and the Iris dataset lab.

Module 3: Supervised Learning

6-7 hours

Linear regression, logistic regression, decision trees, ensemble methods (Random Forest, XGBoost), regularization, and housing price prediction assignment.

Module 4: Neural Networks

7-8 hours

Biological inspiration, the perceptron, multi-layer networks, backpropagation, activation functions, and building networks with TensorFlow/Keras.

Module 5: Deep Learning

8-9 hours

CNNs for computer vision, RNNs for sequences, regularization techniques (dropout, batch norm), transfer learning, and PyTorch introduction.

Module 6: NLP & Transformers

8-10 hours

Text preprocessing, word embeddings (Word2Vec, GloVe), attention mechanisms, Transformer architecture, BERT, and the agentic capstone project.
