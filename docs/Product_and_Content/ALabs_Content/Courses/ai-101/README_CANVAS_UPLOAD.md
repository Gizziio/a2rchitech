## Upload Methods

### Method 1: Node.js Script (Recommended)

Prerequisites:
```bash
npm install @playwright/test
```

Usage:
```bash
node canvas-import.js \
  --email your@email.com \
  --password yourpassword \
  --courseId 12345 \
  --baseUrl https://canvas.instructure.com
```

Or with environment variables:
```bash
export CANVAS_EMAIL=your@email.com
export CANVAS_PASSWORD=yourpassword
export CANVAS_COURSE_ID=12345
node canvas-import.js
```

### Method 2: Python Script

Prerequisites:
```bash
pip install playwright
playwright install chromium
```

Usage:
```bash
python canvas_uploader.py \
  --email your@email.com \
  --password yourpassword \
  --course-id 12345 \
  --base-url https://canvas.instructure.com
```

Or with environment variables:
```bash
export CANVAS_EMAIL=your@email.com
export CANVAS_PASSWORD=yourpassword
export CANVAS_COURSE_ID=12345
python canvas_uploader.py
```

### Method 3: Bash Script with agent-browser

Prerequisites:
- agent-browser installed

Usage:
```bash
./upload_to_canvas.sh your@email.com yourpassword 12345
```

## Manual Upload Instructions

If automated upload fails, follow these manual steps:

### Step 1: Create Pages
1. Navigate to Course → Pages
2. Click "+ Page" for each main page
3. Switch to HTML Editor mode
4. Copy and paste content from HTML files
5. Save and publish

### Step 2: Create Modules
1. Navigate to Course → Modules
2. Click "+ Module" for each module
3. Name modules according to the list above
4. Add items to each module:
   - Pages (link to created pages)
   - Discussions (create new)
   - Assignments (create new)
   - Quizzes (create new)

### Step 3: Organize Content
1. Arrange module items in logical order
2. Set prerequisites where needed
3. Publish all content

## Troubleshooting

### Login Issues
- Verify credentials are correct
- Check if 2FA is enabled (may need manual intervention)
- Ensure Canvas instance URL is correct

### Content Upload Issues
- Check HTML file sizes (Canvas has limits)
- Verify HTML is valid
- Try splitting large content into smaller pages

### Script Errors
- Ensure all dependencies are installed
- Check browser is not in headless mode for debugging
- Review error screenshots generated by scripts
