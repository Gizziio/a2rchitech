Core Pattern (Normalized for allternit)

We normalize the MD into a single canonical primitive:

🔹 Intent Graph (IG)

Intent Graph =
A persistent, typed, journal-backed graph of provisional → committed intent, with:
    •    relational edges (depends_on, blocks, derived_from)
    •    temporal overlays (now / next / later)
    •    contextual provenance (who/where/why)
    •    multiple projections (UI is a view, not the graph)

This is not a new store.
It is a semantic organization of journaled artifacts.
