# Allternit Dynamic Discovery UI  
## Web Capsules, Memory, Skills, and Agentic Web Use

---

## 0. Core Reframe (Non-Negotiable)
Allternit is **not a browser**.

It is a **discovery-first agentic operating environment** where:
- Intent precedes navigation
- Web pages are **spawned artifacts**, not destinations
- Memory compresses exploration into durable intelligence
- Skills emerge visually from lived interaction
- Capsules are the atomic unit of experience

Browsing is a *consequence* of discovery, not the primary mode.

---

## 1. Capsules as the Universal Primitive

### Definition
A **Capsule** is a stateful, executable miniapp with:
- UI
- Memory hooks
- Transition semantics
- Agent interoperability

Everything is a capsule:
- Searches
- Web pages
- Snippets
- Skills
- Distillations
- Replays
