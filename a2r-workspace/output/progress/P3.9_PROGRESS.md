### ✅ P3.9.2 API Layer
- 6 REST endpoints in `mcp_apps_routes.rs`:
  - `POST /mcp-apps/capsules` - Create capsule
  - `GET /mcp-apps/capsules/:id` - Get capsule state
  - `POST /mcp-apps/capsules/:id/event` - Post event
  - `GET /mcp-apps/capsules/:id/stream` - SSE event stream
  - `DELETE /mcp-apps/capsules/:id` - Delete capsule
- SSE streaming for real-time updates
- Compiles successfully ✅

### ✅ P3.9.3 CapsuleFrame Component
- Sandboxed iframe with strict CSP
- `window.a2r` API injection:
  - `invokeTool()` - Call tools from capsule
  - `emitEvent()` - Send events to parent
  - `subscribe()` - Listen to state changes
  - `getState()` - Access current state
- React version (compatible with a2r-platform)
- Connection status indicator
- Error boundaries
