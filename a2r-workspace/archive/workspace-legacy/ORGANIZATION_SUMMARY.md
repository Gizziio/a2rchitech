The following types of documents have been moved to the `finished/` directory:

- Roadmap documents

## Ongoing Work
The following types of documents have been moved to the `ongoing/` directory:

- Feature specifications
- Integration documents
- Session summaries
- Agent documentation
- Quickstart guides
- Release workflows
- Other work-in-progress documentation

## Purpose
This organization separates completed, stable documentation from documents that are still evolving as part of active development. The `finished/` directory contains frozen specifications and completion certificates, while the `ongoing/` directory contains documents that are still being refined and updated as the system evolves.
