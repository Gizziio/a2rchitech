## Core API

### HooksSystemEngine

```rust
use a2r_hooks_system::{HooksSystemEngine, HookDefinition, HookLayer, ToolType, SafetyTier};
use a2rchitech_system_law::SystemLawEngine;
use a2rchitech_harness_engineering::HarnessEngineeringEngine;

// Create engine with SYSTEM_LAW + Harness integration
let system_law = Arc::new(SystemLawEngine::new());
let harness = Arc::new(HarnessEngineeringEngine::new(system_law.clone()));
let hooks = HooksSystemEngine::new(system_law, harness);

// Register hook
let hook = HookDefinition {
    hook_id: "tool_gate".to_string(),
    name: "Tool Gate".to_string(),
    layer: HookLayer::Kernel,
    event_types: vec!["tool_invoke".to_string()],
    handler: "tool_gate_handler".to_string(),
    priority: 1,
    blocking: true,
    enabled: true,
};
hooks.register_hook(hook).await;

// Kernel hooks
hooks.on_boot("init", "kernel").await;
hooks.on_tool_invoke("fs.read", ToolType::Read, SafetyTier::Safe).await;

// Workspace hooks
hooks.on_system_law_check().await;
hooks.on_spec_check("Vision.md").await;

// Task hooks
hooks.on_task_output("task_001", "code", true).await;

// Human layer hooks
hooks.on_escalation(EscalationLevel::Agent, EscalationLevel::Human, "Complex").await;
hooks.on_approval("merge", true, "user_001").await;

// Habit promotion
let habit_id = hooks.register_habit("Code Review").await;
hooks.run_determinism_test(&habit_id).await?;
hooks.promote_habit(&habit_id, "user", "Validated").await?;
```

## Habit Promotion Protocol

### 6 Stages

| Stage | Description | Requirements |
|-------|-------------|--------------|
| **Documentation** | Informal description | None |
| **Checklist** | Explicit steps | Documentation complete |
| **Template** | Reusable pattern | Checklist validated |
| **Skill** | Formalized procedure | Template tested |
| **Hook** | Automated enforcement | Determinism test passed |
| **HardGate** | Non-bypassable | Failure mode analysis, rollback plan |

### Promotion Flow

```
Documentation → Checklist → Template → Skill → Hook → HardGate
                      ↑
              Determinism Test Required
```
