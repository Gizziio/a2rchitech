## Usage

### Enable Agentation
```bash
npm run dev
# Agentation automatically enabled in dev mode
```

### Toggle
- Press `A` key
- Or click 🎨 button

### Annotate
1. Click 🎯 Select
2. Click element
3. Add notes
4. Click 💾 Save & Copy

### Send to Agent
- Formatted output auto-copied to clipboard
- Paste into A2R coding agent
