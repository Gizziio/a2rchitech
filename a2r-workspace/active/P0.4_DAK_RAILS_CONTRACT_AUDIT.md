**Request:**
```http
GET /v1/work/discover?role=builder&priority=1&ready=true
```

**Response:**
```json
{

      "request_id": "wr_abc123",

  ]
}
```

```http

{
  "extend_seconds": 300
}
```

**Response:**
```json
{

    "expires_at": "2026-02-20T12:10:00Z",
    "created_at": "2026-02-20T12:00:00Z"

}
```
