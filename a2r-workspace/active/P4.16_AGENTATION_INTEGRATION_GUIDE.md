## What is Agentation?

**Agentation** (agentation.com) is a **visual annotation overlay** for UI development:

- Click any UI element to attach comments
- Get structured output with selectors + context
- Copy/paste instructions for coding agents
- **Dev-only** - never ships to production

```typescript
/**
 * A2R Agentation Adapter
 * 
 * Wraps Agentation output with A2R execution context
 * for DAG-ready work items.
 */

export interface AgentationOutput {
  notes: string;
  selectors: string[];
  context: string;
  screenshot?: string;
}

export interface A2RExecutionHeader {
  // UI Surface location
  uiSurface: 'primitives' | 'shell' | 'platform';
  
  // Story context
  storyId?: string;
  storyName?: string;
  componentPath?: string;
  
  // Renderer target
  renderer: 'HUMAN' | 'AGENT';
  
  // Viewport preset
  viewport: {
    width: number;
    height: number;
    device?: 'desktop' | 'tablet' | 'mobile';
  };
  
  // Acceptance criteria
  acceptance: string;
  
  // WIH binding (for DAG execution)
  wihId?: string;
  dagNodeId?: string;
}

export interface A2RAgentationOutput extends AgentationOutput {
  header: A2RExecutionHeader;
  formattedForAgent: string;
}

/**
 * Wrap Agentation output with A2R execution context
 */
export function wrapForA2R(
  output: AgentationOutput,
  header: A2RExecutionHeader
): A2RAgentationOutput {
  const formattedForAgent = formatAgentInstructions(output, header);
  
  return {
    ...output,
    header,
    formattedForAgent,
  };
}

/**
 * Format as copy/paste ready instructions for coding agent
 */
function formatAgentInstructions(
  output: AgentationOutput,
  header: A2RExecutionHeader
): string {
  return `
# A2R UI Annotation → Agent Instructions

## Execution Context
| Field | Value |
|-------|-------|
| **UI Surface** | ${header.uiSurface} |
| **Story** | ${header.storyId || 'N/A'} (${header.storyName || 'Unnamed'}) |
| **Component** | ${header.componentPath || 'N/A'} |
| **Renderer** | ${header.renderer} |
| **Viewport** | ${header.viewport.width}x${header.viewport.height}${header.viewport.device ? ` (${header.viewport.device})` : ''} |
| **WIH ID** | ${header.wihId || 'TBD'} |
| **DAG Node** | ${header.dagNodeId || 'TBD'} |

## Acceptance Criteria
${header.acceptance}

---

## Agentation Notes
${output.notes}

## Target Selectors
${output.selectors.map(s => `- \`${s}\``).join('\n')}

## Context
${output.context}

${output.screenshot ? `## Screenshot\n![Annotation](${output.screenshot})\n` : ''}
---

## Agent Action Required

1. **Locate component** using selectors above
2. **Review notes** for desired changes
3. **Implement changes** in component file
4. **Run Storybook** to verify visual changes
5. **Run interaction tests** to verify functionality
6. **Update snapshot** if visual changes are expected

## Verification Commands

\`\`\`bash
# Run Storybook for this component
npm run dev -w @repo/storybook

# Run interaction tests
npm run test:interaction -w @repo/storybook

# Build Storybook (production check)
npm run build -w @repo/storybook
\`\`\`
`.trim();
}

/**
 * Parse Agentation clipboard output
 * (Agentation copies JSON to clipboard when you click "Copy")
 */
export function parseAgentationClipboard(clipboardText: string): AgentationOutput {
  try {
    const parsed = JSON.parse(clipboardText);
    return {
      notes: parsed.notes || '',
      selectors: parsed.selectors || [],
      context: parsed.context || '',
      screenshot: parsed.screenshot,
    };
  } catch (error) {
    throw new Error(
      `Invalid Agentation clipboard format. Expected JSON, got: ${clipboardText.substring(0, 100)}...`
    );
  }
}

/**
 * Quick helper for common use case
 */
export function createA2RWorkItem(
  clipboardText: string,
  header: Omit<A2RExecutionHeader, 'wihId' | 'dagNodeId'>
): A2RAgentationOutput {
  const output = parseAgentationClipboard(clipboardText);
  return wrapForA2R(output, {
    ...header,
    wihId: 'TBD',
    dagNodeId: 'TBD',
  });
}
```

### Step 6: Create CLI Helper Script (2 hours)

**File:** `scripts/wrap-agentation-output.ts`

```typescript
#!/usr/bin/env tsx

/**
 * CLI helper to wrap Agentation clipboard output
 * 
 * Usage:
 *   npm run agentation:wrap -- --story="Button/Primary" --surface="primitives"
 * 
 * Then paste Agentation clipboard output when prompted.
 */

import { wrapForA2R, parseAgentationClipboard } from '../src/lib/agentation-adapter';
import * as readline from 'readline';

interface Args {
  story?: string;
  surface?: string;
  renderer?: 'HUMAN' | 'AGENT';
  width?: number;
  height?: number;
}

function parseArgs(): Args {
  const args: Args = {};
  const argv = process.argv.slice(2);
  
  for (let i = 0; i < argv.length; i++) {
    if (argv[i] === '--story' && argv[i + 1]) {
      args.story = argv[++i];
    } else if (argv[i] === '--surface' && argv[i + 1]) {
      args.surface = argv[++i];
    } else if (argv[i] === '--renderer' && argv[i + 1]) {
      args.renderer = argv[++i] as 'HUMAN' | 'AGENT';
    } else if (argv[i] === '--width' && argv[i + 1]) {
      args.width = parseInt(argv[++i], 10);
    } else if (argv[i] === '--height' && argv[i + 1]) {
      args.height = parseInt(argv[++i], 10);
    }
  }
  
  return args;
}

async function main() {
  const args = parseArgs();
  
  console.log('🎨 A2R Agentation Wrapper\n');
  console.log('Paste Agentation clipboard output (JSON), then press Enter:\n');
  
  const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout,
  });
  
  rl.question('> ', (clipboardText) => {
    try {
      const output = parseAgentationClipboard(clipboardText);
      
      const header = {
        uiSurface: (args.surface as any) || 'primitives',
        storyId: args.story,
        storyName: args.story?.split('/').pop(),
        componentPath: undefined,
        renderer: args.renderer || 'AGENT',
        viewport: {
          width: args.width || 1280,
          height: args.height || 720,
          device: args.width && args.width < 768 ? 'mobile' : args.width && args.width < 1024 ? 'tablet' : 'desktop',
        },
        acceptance: 'Storybook build passes + no visual regressions',
        wihId: 'TBD',
        dagNodeId: 'TBD',
      };
      
      const result = wrapForA2R(output, header);
      
      console.log('\n✅ Wrapped for A2R Agent:\n');
      console.log(result.formattedForAgent);
      console.log('\n📋 Copied to clipboard!');
      
    } catch (error) {
      console.error('❌ Error:', error instanceof Error ? error.message : error);
    } finally {
      rl.close();
    }
  });
}

main();
```

# Agentation for A2R

## What is Agentation?

Agentation is a visual annotation tool that lets you:
1. Click any UI element in Storybook
2. Add notes describing desired changes
3. Get structured output with selectors + context
4. Copy/paste into A2R coding agent

## Quick Start

### 1. Enable Agentation

```bash
# In storybook directory
npm run dev:agentation
```

Or set environment variable:
```bash
export A2R_AGENTATION=true
```

### 2. Open Storybook

Navigate to the story you want to annotate:
```
http://localhost:6006/?path=/story/button--primary
```

### 3. Toggle Agentation

- Click the Agentation icon (bottom-right corner)
- Or press `A` key (default hotkey)

### 4. Annotate

1. Click any element to select it
2. Add your notes
3. Click "Copy Output"

### 5. Wrap for A2R

Run the wrapper script:
```bash
npm run agentation:wrap -- --story="Button/Primary" --surface="primitives"
```

Paste the clipboard output when prompted.

### 6. Send to Agent

Copy the formatted output and paste into your A2R coding agent (e.g., UI_IMPLEMENTER).

## Output Format

The wrapper produces:

```markdown
# A2R UI Annotation → Agent Instructions

## Execution Context
| Field | Value |
|-------|-------|
| UI Surface | primitives |
| Story | Button/Primary |
| Renderer | AGENT |
| Viewport | 1280x720 (desktop) |

## Acceptance Criteria
Storybook build passes + no visual regressions

## Agentation Notes
[Your notes here]

## Target Selectors
- `[data-storybook="Button"]`
- `.button-primary`

## Agent Action Required
1. Locate component using selectors
2. Review notes for changes
3. Implement changes
4. Run Storybook to verify
5. Run interaction tests
6. Update snapshot if needed
```

## Best Practices

### DO
- ✅ Annotate in isolated Storybook stories
- ✅ Be specific about desired changes
- ✅ Include acceptance criteria
- ✅ Test with interaction tests after changes

### DON'T
- ❌ Annotate in production builds (Agentation is dev-only)
- ❌ Annotate multiple components at once
- ❌ Forget to run tests after implementing changes

## Troubleshooting

### Agentation not loading
- Check `A2R_AGENTATION=true` environment variable
- Check browser console for errors
- Try hard refresh (Cmd+Shift+R)

### Invalid clipboard format
- Make sure you clicked "Copy Output" in Agentation
- Clipboard should contain JSON, not formatted text

### Selectors not working
- Storybook may use different DOM structure than app
- Verify selectors in browser DevTools
- Consider adding `data-testid` attributes

## Integration with A2R DAG

The wrapped output can be used as input for:
- `UI_IMPLEMENTER` agent role
- `UI_ARCHITECT` agent role
- `UI_TESTER` agent role

Each role has different permissions:
- **IMPLEMENTER**: Can write code, cannot modify specs
- **ARCHITECT**: Can modify specs, cannot write code
- **TESTER**: Can write tests, cannot write production code
