**Features Implemented:**

- ✅ Worker manager integration (spawn/track workers)
- ✅ Heartbeat client (30-second intervals with health metrics)
- ✅ Tool execution layer (poll for tasks, execute, report results)
- ✅ Installation scripts (CLI with clap)
- ✅ Graceful shutdown (Ctrl+C handling)

**CLI Usage:**
```bash
# Run with environment variables
export A2R_CONTROL_PLANE_URL="https://control.a2r.cloud"
export A2R_NODE_SECRET="your-secret"
export A2R_REGION="us-west"
cargo run -p a2r-edge-runner

# Or with command line arguments
cargo run -p a2r-edge-runner -- \
  --control-plane-url https://control.a2r.cloud \
  --node-secret your-secret \
  --region us-west \
  --max-workers 10
```

**Configuration:**
| Option | Env Var | Default | Description |
|--------|---------|---------|-------------|
| `--control-plane-url` | `A2R_CONTROL_PLANE_URL` | https://control.a2r.cloud | Control plane endpoint |
| `--node-secret` | `A2R_NODE_SECRET` | (required) | Node authentication secret |
| `--node-id` | `A2R_NODE_ID` | auto-generated | Node ID |
| `--node-name` | `A2R_NODE_NAME` | edge-runner | Node name |
| `--heartbeat-interval` | `A2R_HEARTBEAT_INTERVAL` | 30 | Heartbeat interval (seconds) |
| `--max-workers` | `A2R_MAX_WORKERS` | 10 | Maximum concurrent workers |
| `--region` | `A2R_REGION` | us-west | Region |
| `--zone` | `A2R_ZONE` | (optional) | Zone |
