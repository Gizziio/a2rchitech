## Six GC Agents (Daily Execution)

### 1. Duplicate Detector
**Purpose:** Detect duplicate code utilities

**Detects:**
- Repeated code blocks (5+ lines)
- Similar function patterns
- Duplicate imports

**Severity:** Warning

---

### 2. Boundary Type Checker
**Purpose:** Detect untyped boundary usage

**Detects:**
- `.unwrap()` calls
- `.expect()` calls
- `panic!` in production code

**Severity:** Warning

---

### 3. Dependency Validator
**Purpose:** Detect dependency violations

**Detects:**
- Layer violations (e.g., UI importing kernel)
- Circular dependencies
- Forbidden imports

**Severity:** Error

---

### 4. Observability Checker
**Purpose:** Detect missing observability

**Detects:**
- Public functions without tracing
- Missing structured logging
- Missing metrics

**Severity:** Info

---

### 5. Doc Staleness Checker
**Purpose:** Detect stale documentation

**Detects:**
- TODO comments
- FIXME comments
- Outdated documentation

**Severity:** Warning

---

### 6. Test Coverage Checker
**Purpose:** Detect test coverage gaps

**Detects:**
- Modules without tests
- Missing unit tests
- Missing integration tests

**Severity:** Warning

## Entropy Score (LAW-QLT-002)

**Calculation:**
```
Entropy Score = (Critical × 10) + (Warning × 5) + (Info × 1)
```

**Target:** < 5% drift/month

**Tracking:**
- Current score
- Historical records
- Total reduction from GC runs

## Core API

```rust
use a2r_gc_agents::{GcAgentOrchestrator, GcAgentConfig, GcReportGenerator};

// Create orchestrator
let config = GcAgentConfig {
    root_path: PathBuf::from("/path/to/codebase"),
    exclude_patterns: vec![
        "**/target/**".to_string(),
        "**/node_modules/**".to_string(),
    ],
    auto_fix: false,
    dry_run: true,
};

let orchestrator = GcAgentOrchestrator::new(config);

// Run all GC agents
let results = orchestrator.run_all().await?;

// Get entropy score
let score = orchestrator.get_entropy_score().await;
println!("Current entropy score: {}", score.score);

// Generate report
let reporter = GcReportGenerator::new();
let markdown = reporter.generate_markdown(&results);
reporter.save_report(&results, &PathBuf::from("gc-report.md"))?;
```

## Usage Examples

### Daily GC Run (Cron Job)
```rust
#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let config = GcAgentConfig::default();
    let orchestrator = GcAgentOrchestrator::new(config);
    
    let results = orchestrator.run_all().await?;
    
    // Generate and save report
    let reporter = GcReportGenerator::new();
    reporter.save_report(&results, &PathBuf::from("/var/log/gc-report.md"))?;
    
    Ok(())
}
```

### Run Specific Agent
```rust
let orchestrator = GcAgentOrchestrator::new(GcAgentConfig::default());

// Run only the duplicate detector
let result = orchestrator.run_agent("duplicate_detector").await?;
println!("Found {} duplicate issues", result.issues_found.len());
```

### Check Entropy Score
```rust
let score = orchestrator.get_entropy_score().await;

if score.score > 100.0 {
    println!("⚠️ High entropy detected: {}", score.score);
    println!("  Critical issues: {}", score.critical_issues);
    println!("  Warning issues: {}", score.warning_issues);
}
```
