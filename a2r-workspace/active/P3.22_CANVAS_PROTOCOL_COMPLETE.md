## Core API

### CanvasRuntime

```rust
use a2r_canvas_protocol::{CanvasRuntime, CanvasSpec, CanvasViewType, CanvasBindings};
use a2rchitech_system_law::SystemLawEngine;

// Create runtime with SYSTEM_LAW integration
let system_law = Arc::new(SystemLawEngine::new());
let runtime = CanvasRuntime::new(system_law);

// Create canvas
let spec = CanvasSpec {
    canvas_id: "cnv_001".to_string(),
    view_type: CanvasViewType::DiffView,
    title: Some("Diff Review".to_string()),
    bindings: CanvasBindings {
        run_id: Some("run_001".to_string()),
        journal_refs: vec!["jrnl_001".to_string()],
        artifact_refs: vec!["art_diff_001".to_string()],
        repo_snapshot_ref: Some("git:abc123".to_string()),
    },
    // ... other fields
};

let canvas_id = runtime.create_canvas(spec).await?;

// Get canvas
let canvas = runtime.get_canvas(&canvas_id).await;

// Update canvas via patch
let patch = CanvasPatch {
    bindings: Some(new_bindings),
    interactions: None,
    filters: None,
};
runtime.update_canvas(&canvas_id, patch).await?;

// List canvases by type
let diff_canvases = runtime.list_canvases_by_type(CanvasViewType::DiffView).await;

// Destroy canvas
runtime.destroy_canvas(&canvas_id).await?;
```

## SYSTEM_LAW Compliance

| LAW Section | Implementation |
|-------------|----------------|
| **LAW-ONT-001** (Canonical Definitions) | CanvasSpec with journal bindings required |
| **LAW-ONT-003** (Determinism) | Risk must be declared for write/exec |

- Canvas must have at least one journal reference
- Risk reason required for write/exec canvases
