# Integrate with Registry Service

## Objective
Integrate skill portability with the Registry Service.

## API Endpoints to Add

### `POST /api/v1/skills`
Publish a skill to the registry:
```json
{
  "name": "my-skill",
  "content": "...",
  "metadata": {...}
}
```

### `GET /api/v1/skills`
List available skills:
```json
[
  {
    "id": "skill-123",
    "name": "my-skill",
    "description": "..."
  }
]
```

### `POST /api/v1/skills/{id}/install`
Install a skill to specified LLMs:
```json
{
  "llms": ["claude", "codex"],
  "scope": "global"
}
```

## Implementation
- Extend Registry Service with skill storage
- Add API routes in Gateway
- Integrate with skill-portability crate
