# Implement Branch Auto-Creation

## Objective
Automatically create and manage branches for DAG nodes.

## Branch Naming
```rust
pub fn generate_branch_name(dag_id: &str, node_id: &str) -> String {
    format!("a2r/{}/{}", sanitize(dag_id), sanitize(node_id))
}

// Examples:
// a2r/D2026-02-13/N1
// a2r/feature-auth/N2.1
```

## BranchManager
```rust
pub struct BranchManager;

impl BranchManager {
    pub async fn ensure_branch(&self, name: &str) -> Result<Branch>;
    pub async fn create_from_base(&self, name: &str, base: &str) -> Result<()>;
    pub async def delete_branch(&self, name: &str) -> Result<()>;
}
```

## Features
- Generate unique branch names
- Create from default branch
- Handle existing branches
- Track branch-to-node mapping
