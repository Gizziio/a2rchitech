## Objective
Create a new Rust crate in the 0-substrate layer for skill portability across LLMs.

## Crate Structure
```
a2r-skill-portability/
├── Cargo.toml
├── src/
│   ├── lib.rs
│   ├── types.rs
│   ├── drivers/
```
