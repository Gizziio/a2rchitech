# Implement Claude Code Driver

## Objective
Implement the SkillDriver trait for Claude Code integration.

## Implementation Details

### Target Directory
- `~/.claude/commands/` for global skills
- `.claude/commands/` for workspace-local skills

### Settings Integration
Claude Code uses `settings.json` for configuration:
```json
{
  "commands": {
    "my-skill": {
      "description": "Skill description",
      "prompt": "path/to/skill.md"
    }
  }
}
```

### Methods to Implement
1. `install_skill()` - Copy skill to commands dir, update settings
2. `remove_skill()` - Remove skill file, update settings
3. `list_skills()` - Read commands directory
4. `skill_format()` - Return Markdown format
