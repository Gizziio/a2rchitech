# Implement Cleanup Policies

## Objective
Implement worktree cleanup policies.

## Policy Implementation

### OnDone
```rust
pub async fn on_node_done(&self, node: &DagNode) -> Result<()> {
    if let Some(config) = &node.worktree {
        if config.cleanup_on_done == CleanupPolicy::OnDone {
            self.remove_worktree(&node.worktree_path()).await?;
        }
    }
    Ok(())
}
```

### OnDagComplete
```rust
pub async fn on_dag_complete(&self, dag: &DagState) -> Result<()> {
    for node in &dag.nodes {
        if let Some(config) = &node.worktree {
            if config.cleanup_on_done == CleanupPolicy::OnDagComplete {
                self.remove_worktree(&node.worktree_path()).await?;
            }
        }
    }
    Ok(())
}
```

### Never
- Keep worktree indefinitely
- Manual cleanup required
