# Add Recovery UI to Shell

## Objective
Create UI for session recovery in the Shell.

## Components

### SnapshotBrowser
```tsx
interface SnapshotBrowserProps {
  snapshots: Snapshot[];
  onRestore: (snapshot: Snapshot) => void;
  onDelete: (snapshot: Snapshot) => void;
}

export const SnapshotBrowser: React.FC<SnapshotBrowserProps> = ({
  snapshots,
  onRestore,
  onDelete,
}) => {
  // Show list of snapshots
  // Allow restore/delete
  // Show snapshot details
};
```

### SnapshotCard
```tsx
interface SnapshotCardProps {
  snapshot: Snapshot;
  onRestore: () => void;
  onDelete: () => void;
}
```

## Features
- Browse snapshots
- View snapshot details (DAG state, terminals, worktrees)
- Restore to snapshot
- Delete old snapshots
- Auto-snapshot settings
