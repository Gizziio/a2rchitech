# Implement Session Manager

## Objective
Manage tmux sessions with persistent state.

## SessionManager
```rust
pub struct SessionManager {
    tmux: TmuxClient,
    state: Arc<Mutex<SessionState>>,
}

impl SessionManager {
    pub async fn create(&self, config: SessionConfig) -> Result<Session>;
    pub async fn get(&self, id: &str) -> Result<Session>;
    pub async fn list(&self) -> Result<Vec<Session>>;
    pub async fn kill(&self, id: &str) -> Result<()>;
    pub async fn attach(&self, id: &str) -> Result<AttachHandle>;
}
```

## SessionState
```rust
pub struct SessionState {
    sessions: HashMap<String, Session>,
    panes: HashMap<String, Pane>,
}
```

## Features
- Track active sessions
- Map sessions to DAG nodes
- Persist session metadata
- Handle tmux crashes/restarts
