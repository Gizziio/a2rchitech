# Implement Tmux Client Library

## Objective
Create a Rust wrapper around tmux commands.

## API Design

### TmuxClient
```rust
pub struct TmuxClient;

impl TmuxClient {
    pub fn new_session(name: &str) -> Result<Session>;
    pub fn kill_session(name: &str) -> Result<()>;
    pub fn list_sessions() -> Result<Vec<Session>>;
    
    pub fn split_window(session: &str, command: &str) -> Result<Pane>;
    pub fn send_keys(pane_id: &str, keys: &str) -> Result<()>;
    pub fn capture_pane(pane_id: &str) -> Result<String>;
}
```

### Types
```rust
pub struct Session {
    pub name: String,
    pub windows: Vec<Window>,
    pub created_at: DateTime,
}

pub struct Pane {
    pub id: String,
    pub session: String,
    pub window: u32,
    pub command: Option<String>,
}
```

## Implementation Notes
- Use `tmux` binary directly via std::process
- Parse tmux's formatted output
- Handle errors gracefully
