# Add Recovery Commands to CLI

## Objective
Add CLI commands for session recovery.

## Commands

### `a2r session snapshot`
Create a snapshot of the current session:
```bash
a2r session snapshot
a2r session snapshot --name "before-refactor"
```

### `a2r session restore`
Restore from a snapshot:
```bash
a2r session restore <snapshot-id>
a2r session restore --latest
```

### `a2r session list-snapshots`
List available snapshots:
```bash
a2r session list-snapshots
a2r session list-snapshots --session <id>
```

## Implementation
```rust
pub fn session_subcommand() -> Command {
    Command::new("session")
        .subcommand(Command::new("snapshot").arg(arg!(--name <NAME>)))
        .subcommand(Command::new("restore").arg(arg!(<SNAPSHOT_ID>)))
        .subcommand(Command::new("list-snapshots"))
}
```
