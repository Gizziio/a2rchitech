# Create Session Recovery Service

## Objective
Create a service for snapshotting and recovering sessions.

## Service Structure
```
session-recovery/
├── Cargo.toml
├── src/
│   ├── lib.rs
│   ├── snapshot.rs
│   ├── recovery.rs
│   └── storage.rs
└── tests/
```

## SessionRecovery
```rust
pub struct SessionRecovery {
    ledger: Arc<Ledger>,
    workspace: WorkspaceClient,
    storage: SnapshotStorage,
}

impl SessionRecovery {
    pub async fn snapshot(&self, session_id: &str) -> Result<Snapshot>;
    pub async fn restore(&self, snapshot: &Snapshot) -> Result<Session>;
    pub async fn list_snapshots(&self) -> Result<Vec<Snapshot>>;
}
```

pub struct Snapshot {
