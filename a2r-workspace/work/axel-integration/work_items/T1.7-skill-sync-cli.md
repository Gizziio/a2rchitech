# Add Skill Sync Commands to CLI

## Objective
Add CLI commands for skill portability management.

## Commands to Implement

### `a2r skill sync`
Sync skills across all configured LLMs:
```bash
a2r skill sync --llm claude --llm codex
a2r skill sync --all
```

### `a2r skill install <path>`
Install a skill from file or directory:
```bash
a2r skill install ./skills/my-skill.md
a2r skill install ./skills/ --all
```

### `a2r skill list`
List installed skills:
```bash
a2r skill list
a2r skill list --llm claude
```

### `a2r skill remove <name>`
Remove a skill:
```bash
a2r skill remove my-skill
a2r skill remove my-skill --llm claude
```

## Implementation
- Use clap for argument parsing
- Integrate with a2r-skill-portability crate
- Support both global and workspace-local skills
