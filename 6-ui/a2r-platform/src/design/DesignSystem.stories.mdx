# A2rchitect Design System

The A2R Design System provides a comprehensive set of visual standards, components, and patterns for building consistent user interfaces across the platform.

## Philosophy

- **Glass Morphism**: Translucent, layered interfaces with depth
- **Motion-First**: Smooth, purposeful animations enhance UX
- **Accessibility**: WCAG 2.1 AA compliance built-in
- **Themeable**: Light and dark modes with seamless switching

## Motion

### Timing Functions

- **Fast**: 140ms cubic-bezier(0.4, 0, 0.2, 1) - Micro-interactions
- **Base**: 200ms cubic-bezier(0.4, 0, 0.2, 1) - Standard transitions
- **Slow**: 300ms cubic-bezier(0.4, 0, 0.2, 1) - Emphasis animations
- **Spring**: cubic-bezier(0.34, 1.56, 0.64, 1) - Playful bounces

### Animation Principles

1. **Purposeful**: Every animation serves a functional purpose
2. **Performant**: GPU-accelerated transforms and opacity only
3. **Respectful**: Honors `prefers-reduced-motion`
4. **Consistent**: Same timing across similar interactions

## Glass System

### Intensity Levels

| Level | Blur | Background | Use Case |
|-------|------|------------|----------|
| Thin  | 16px | var(--glass-bg) | Tooltips, dropdowns |
| Base  | 24px | var(--glass-bg) | Cards, panels |
| Elevated | 32px | var(--glass-bg-thick) | Modals, overlays |
| Thick | 40px | var(--glass-bg-thick) | Full-screen overlays |

### CSS Variables

```css
--glass-bg: rgba(255, 255, 255, 0.05);
--glass-bg-thick: rgba(30, 30, 40, 0.6);
--border-subtle: rgba(255, 255, 255, 0.08);
--border-default: rgba(255, 255, 255, 0.12);
--border-strong: rgba(255, 255, 255, 0.18);
```
