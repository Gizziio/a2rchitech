# EmailcomposercardPlugin Plugin

Based on `EmailComposerCard.tsx` template.

## Description

Plugin based on EmailComposerCard.tsx template

## Installation

```bash
npm install @allternit/emailcomposercard-plugin
```

## Usage

```bash
allternit-plugin run emailcomposercard-plugin --input "your input"
```

## Template Source

Original template: `EmailComposerCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd emailcomposercard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
