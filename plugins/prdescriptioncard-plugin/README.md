# PrdescriptioncardPlugin Plugin

Based on `PRDescriptionCard.tsx` template.

## Description

Plugin based on PRDescriptionCard.tsx template

## Installation

```bash
npm install @allternit/prdescriptioncard-plugin
```

## Usage

```bash
allternit-plugin run prdescriptioncard-plugin --input "your input"
```

## Template Source

Original template: `PRDescriptionCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd prdescriptioncard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
