# TranslationcardPlugin Plugin

Based on `TranslationCard.tsx` template.

## Description

Plugin based on TranslationCard.tsx template

## Installation

```bash
npm install @allternit/translationcard-plugin
```

## Usage

```bash
allternit-plugin run translationcard-plugin --input "your input"
```

## Template Source

Original template: `TranslationCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd translationcard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
