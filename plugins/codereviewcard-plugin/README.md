# CodereviewcardPlugin Plugin

Based on `CodeReviewCard.tsx` template.

## Description

Plugin based on CodeReviewCard.tsx template

## Installation

```bash
npm install @allternit/codereviewcard-plugin
```

## Usage

```bash
allternit-plugin run codereviewcard-plugin --input "your input"
```

## Template Source

Original template: `CodeReviewCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd codereviewcard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
