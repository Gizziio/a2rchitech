# ChatbotcardPlugin Plugin

Based on `ChatbotCard.tsx` template.

## Description

Plugin based on ChatbotCard.tsx template

## Installation

```bash
npm install @allternit/chatbotcard-plugin
```

## Usage

```bash
allternit-plugin run chatbotcard-plugin --input "your input"
```

## Template Source

Original template: `ChatbotCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd chatbotcard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
