# SocialmediacardPlugin Plugin

Based on `SocialMediaCard.tsx` template.

## Description

Plugin based on SocialMediaCard.tsx template

## Installation

```bash
npm install @allternit/socialmediacard-plugin
```

## Usage

```bash
allternit-plugin run socialmediacard-plugin --input "your input"
```

## Template Source

Original template: `SocialMediaCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd socialmediacard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
