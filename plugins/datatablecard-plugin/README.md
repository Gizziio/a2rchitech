# DatatablecardPlugin Plugin

Based on `DataTableCard.tsx` template.

## Description

Plugin based on DataTableCard.tsx template

## Installation

```bash
npm install @allternit/datatablecard-plugin
```

## Usage

```bash
allternit-plugin run datatablecard-plugin --input "your input"
```

## Template Source

Original template: `DataTableCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd datatablecard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
