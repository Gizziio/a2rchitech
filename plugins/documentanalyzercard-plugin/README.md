# DocumentanalyzercardPlugin Plugin

Based on `DocumentAnalyzerCard.tsx` template.

## Description

Plugin based on DocumentAnalyzerCard.tsx template

## Installation

```bash
npm install @allternit/documentanalyzercard-plugin
```

## Usage

```bash
allternit-plugin run documentanalyzercard-plugin --input "your input"
```

## Template Source

Original template: `DocumentAnalyzerCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd documentanalyzercard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
