# ApispeccardPlugin Plugin

Based on `APISpecCard.tsx` template.

## Description

Plugin based on APISpecCard.tsx template

## Installation

```bash
npm install @allternit/apispeccard-plugin
```

## Usage

```bash
allternit-plugin run apispeccard-plugin --input "your input"
```

## Template Source

Original template: `APISpecCard.tsx`
See `docs/original-template.txt` for reference.

## Development

```bash
cd apispeccard-plugin
npm install
npm run build
```

## Adapting the Template

1. Review `docs/original-template.txt`
2. Adapt the logic in `src/index.ts`
3. Update manifest.json with correct parameters
4. Test with `allternit-plugin run`
