# Allternit Skills Package

The skills package provides the skill packaging, manifest parsing, and routing system for Allternit, enabling installable, versioned, and signed capabilities.

## Features

- Skill manifest parsing and validation
- Typed I/O schema validation
- Side effect declaration and enforcement
- Risk tier assignment and validation
- Skill routing and discovery
- Versioning and dependency management
- Signature verification

## Components

- Schema: JSON Schema validation for inputs/outputs
- Packaging: Skill bundle structure and validation
- Versioning: Semantic versioning and compatibility checks
- Routing: Skill discovery and selection
- Evals: Skill testing and benchmarking
