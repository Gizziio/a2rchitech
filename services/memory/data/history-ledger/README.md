# Allternitchitech History Package

The history package provides an append-only JSONL ledger with chained hashes for complete auditability and deterministic replay.

## Features

- Append-only ledger in JSONL format
- Chained hash integrity verification
- Content-addressed storage
- Stream-based access patterns
- Immutable record creation

## Usage

This package is used by the kernel to record all events, tasks, and artifacts for complete auditability and deterministic replay.
