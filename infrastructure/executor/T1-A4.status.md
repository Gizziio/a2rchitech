## Implementation Details

### ExecutorServiceImpl

- **Storage:** `HashMap<JobId, Job>` + `HashMap<JobId, JobStatus>` with `RwLock`
- **submit:** Stores job and sets status to Pending
- **cancel:** Validates state transition, updates to Cancelled
- **status:** Returns current job status or NotFound error
- **list:** Filters and sorts jobs by created_at

### JobQueueServiceImpl

- **Storage:** `BinaryHeap<QueuedJob>` + `HashMap<JobId, Priority>` with `RwLock`
- **enqueue:** Adds to priority queue, returns position (1-indexed)
- **dequeue:** Removes and returns highest priority job
- **peek:** Returns highest priority job without removing
- **remove:** Removes job by ID, rebuilds heap
- **len:** Returns queue length
- **position:** Returns 1-indexed position in queue

### ResourceManagerServiceImpl

- **Storage:** `HashMap<String, Allocation>` with `RwLock`
- **allocate:** Validates resources available, creates allocation
- **release:** Removes allocation, returns error if not found
- **metrics:** Calculates used resources from active allocations

### RuntimeManagerImpl

- **logs:** Spawns tokio task to stream logs via mpsc channel
